  • java.lang.Object
    • randoop.compile.SequenceCompiler

  • public class SequenceCompiler
extends Object
    Compiles a Java class given as a String.

    A simplified version of the javaxtools.compiler.CharSequenceCompiler from Create dynamic applications with javax.tools.
